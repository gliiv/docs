---
title: "Send SMS"
api: "POST https://api.gliiv.com/v1/messages/send"
description: "Send a single or bulk message via the Gliiv gateway."
---

### Authentication
Include your `x-api-key` in the request header as described in the [Authentication](/authentication) guide.

### Body Parameters

<ParamField body="to" type="string" placeholder="+250780000000" required>
  The recipient's phone number in E.164 format.
</ParamField>

<ParamField body="body" type="string" placeholder="Hello from Gliiv!" required>
  The content of your message.
</ParamField>

<ParamField body="senderId" type="string" placeholder="GLIIV">
  The approved Sender ID to be displayed on the recipient's device.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the message was successfully accepted into the queue.
</ResponseField>

<ResponseField name="messageId" type="string">
  A unique identifier for tracking the message status.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.gliiv.com/v1/messages/send \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <your-api-key>' \
  --data '{
    "to": "+250780000000",
    "body": "Your Gliiv verification code is 1234",
    "senderId": "GLIIV"
  }'
```

</RequestExample>
