---

The `/send` endpoint is the core of the Gliiv API. It is designed for high-throughput ingestion, allowing you to send messages to one or multiple recipients in a single request. 

All requests are processed asynchronously to ensure minimal latency for your application.

### Body Parameters

<ParamField body="apiPhoneNumbers" type="string[]" required>
  An array of recipient phone numbers.
  * **Format:** Supports E.164 (`+250...`), local (`07...`), or international formats.
  * **Limit:** Maximum of **20 numbers** per individual API request for optimal performance.
</ParamField>

<ParamField body="message" type="string" required>
  The text content of your SMS.
  * **Billing:** Charged in **160-character segments**.
  * **Maximum:** 800 characters total.
</ParamField>

<ParamField body="senderId" type="string">
  The unique ID of your approved Sender identity.
  * **Note:** This is the internal MongoDB-style ID found in your dashboard (e.g., `69c905148f112344e98b6b94`). If omitted, your default Sender ID will be used.
</ParamField>

### Response Schema

<ResponseField name="success" type="boolean">
  Returns `true` if the message payload passed validation and was accepted into the Gliiv processing queue.
</ResponseField>

<ResponseField name="messageId" type="string">
  A unique tracking ID assigned to this batch. Use this ID to query delivery reports via the status endpoint.
</ResponseField>

<RequestExample>

```javascript Node.js
const axios = require('axios');

const data = {
  apiPhoneNumbers: ["+250700000000"],
  message: "Hello from Gliiv!",
  senderId: "69c905148f112344e98b6b94"
};

const config = {
  method: 'post',
  url: 'https://api.gliiv.com/v1/messages/send',
  headers: { 
    'x-api-key': 'gliiv_xxxxxxxxxxxxxxxx', 
    'Content-Type': 'application/json'
  },
  data : data
};

axios(config)
.then(res => console.log(res.data))
.catch(err => console.error(err));
```

```python Python
import requests
import json

url = "https://api.gliiv.com/v1/messages/send"

payload = json.dumps({
  "apiPhoneNumbers": ["+250700000000"],
  "message": "Hello from Gliiv!",
  "senderId": "69c905148f112344e98b6b94"
})
headers = {
  'x-api-key': 'gliiv_xxxxxxxxxxxxxxxx',
  'Content-Type': 'application/json'
}

response = requests.request("POST", url, headers=headers, data=payload)
print(response.text)
```
