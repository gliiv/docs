---

title: "Send SMS"
api: "POST https://api.gliiv.com/v1/messages/send"
description: "Send high-priority single or bulk messages via the Gliiv gateway."
---

The `/send` endpoint allows you to programmatically dispatch messages to your recipients. All requests are processed through our high-speed gateway to ensure rapid delivery.

<Note>
  **Requirement:** Before using this endpoint, you must have at least one **Sender** assigned to your account.
</Note>

### Body Parameters

<ParamField body="phoneNumbers" type="string[]" required>
  An array of recipient phone numbers. 
  * **Supported Formats:** `+250...`, `250...`, `07...`, or `7...`
  * **Limit:** Maximum of **20 numbers** per individual API request.
</ParamField>

<ParamField body="message" type="string" required>
  The text content of your SMS.
  * **Billing:** Charged in **160-character segments**.
  * **Maximum:** 800 characters total.
</ParamField>

<ParamField body="senderId" type="string" required>
  The unique ID of your assigned **Sender**.
  
  **How to obtain:** 1. Head to the [Senders Tab](https://www.gliiv.com/app/alpha/senders).
  2. Click to preview the details of the specific Sender you wish to use.
  3. Copy the ID using the provided copy button.
</ParamField>

### Response Schema

<ResponseField name="status" type="number">
  The HTTP status code associated with the response.
</ResponseField>

<ResponseField name="message" type="string">
  A descriptive message detailing the outcome of the request.
</ResponseField>

<ResponseField name="validationErrors" type="object">
  **Optional.** This object is only returned on a **400 Bad Request**. It contains key-value pairs where the key is the field name (e.g., `phoneNumbers`, `message`, or `senderId`) and the value is a specific string describing the validation failure for that field.
</ResponseField>

<RequestExample>

```javascript Node.js
const axios = require('axios');

const data = {
  phoneNumbers: ["0780000000"],
  message: "Hello from Gliiv!",
  senderId: "69c905148f112344e98b6b94"
};

const config = {
  method: 'post',
  url: 'https://api.gliiv.com/v1/messages/send',
  headers: { 
    'x-api-key': 'gliiv_xxxxxxxxxxxxxxxx', 
    'Content-Type': 'application/json'
  },
  data : data
};

axios(config)
.then(res => console.log(res.data))
.catch(err => console.error(err.response.data));
```

```bash cURL
curl -X POST https://api.gliiv.com/v1/messages/send \
  -H "x-api-key: <your_api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "phoneNumbers": ["0780000000"],
    "message": "Hello from Gliiv!",
    "senderId": "69c905148f112344e98b6b94"
  }'
```

</RequestExample>

<ResponseExample>

{
    "message": "Messages are being sent",

}
