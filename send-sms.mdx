---
title: "Send SMS"
api: "POST https://api.gliiv.com/v1/messages/send"
description: "Send high-priority single or bulk messages via the Gliiv gateway."
---

The `/send` endpoint allows you to programmatically dispatch messages to Rwandan mobile subscribers. 

<Note>
  **Requirement:** Before using this endpoint, you must have at least one approved **Sender ID** assigned to your account by the Gliiv administration team.
</Note>

### Body Parameters

<ParamField body="apiPhoneNumbers" type="string[]" required>
  An array of recipient phone numbers.
  * **Supported Formats:** Specifically supports Rwandan numbers in E.164 (`+250...`), international (`250...`), or local (`07...` / `7...`) formats.
  * **Limit:** Maximum of **20 numbers** per individual API request.
</ParamField>

<ParamField body="message" type="string" required>
  The text content of your SMS.
  * **Billing:** Charged in **160-character segments**.
  * **Maximum:** 800 characters total.
</ParamField>

<ParamField body="senderId" type="string" required>
  The unique ID of your approved Sender identity.
  
  **How to obtain:**
  1. Go to the [Senders Tab](https://www.gliiv.com/app/alpha/senders).
  2. Click to preview the details of the specific sender you wish to use.
  3. Copy the **Sender ID** using the provided copy button.
</ParamField>

### Response Schema

<ResponseField name="success" type="boolean">
  Returns `true` if the message payload passed validation and was accepted into the processing queue.
</ResponseField>

<ResponseField name="messageId" type="string">
  A unique tracking ID assigned to this batch.
</ResponseField>

<RequestExample>

```javascript Node.js
const axios = require('axios');

const data = {
  apiPhoneNumbers: ["0780000000"],
  message: "Hello from Gliiv!",
  senderId: "69c905148f112344e98b6b94"
};

const config = {
  method: 'post',
  url: 'https://api.gliiv.com/v1/messages/send',
  headers: { 
    'x-api-key': 'gliiv_xxxxxxxxxxxxxxxx', 
    'Content-Type': 'application/json'
  },
  data : data
};

axios(config)
.then(res => console.log(res.data))
.catch(err => console.error(err));
```

```python Python
import requests
import json

url = "https://api.gliiv.com/v1/messages/send"

payload = json.dumps({
  "apiPhoneNumbers": ["0780000000"],
  "message": "Hello from Gliiv!",
  "senderId": "69c905148f112344e98b6b94"
})
headers = {
  'x-api-key': 'gliiv_xxxxxxxxxxxxxxxx',
  'Content-Type': 'application/json'
}

response = requests.request("POST", url, headers=headers, data=payload)
print(response.text)
```

```bash cURL
curl -X POST https://api.gliiv.com/v1/messages/send \
  -H "x-api-key: <your_api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "apiPhoneNumbers": ["0780000000"],
    "message": "Hello from Gliiv!",
    "senderId": "69c905148f112344e98b6b94"
  }'
```
